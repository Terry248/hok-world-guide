# 元素反应机制完全解析

王者荣耀世界中共有7种元素：**光、雷、草、火、水、冰、岩**。不同元素之间相互作用会产生特殊效果。

## 元素反应一览表

| 反应名称 | 触发元素 | 效果 | 伤害倍率 | 适用场景 |
|----------|----------|------|----------|----------|
| **光耀** | 光 + 雷 | 范围感电，对周围敌人造成连锁伤害 | 1.5× | 群怪清场 |
| **燃烧** | 草 + 火 | 持续燃烧，造成持续火伤 | 0.8×/秒 | 控制、削血 |
| **蒸发** | 火 + 水 | 单次高额伤害 | 2.0× | 爆发输出 |
| **冻结** | 水 + 冰 | 冻结敌人，使其无法行动 | 控制 | 控场、打断 |
| **超导** | 雷 + 冰 | 降低敌人防御，造成范围伤害 | 1.2× | 破甲、群攻 |
| **绽放** | 草 + 水 | 生成治疗区域，恢复队伍生命 | 治疗 | 续航 |
| **结晶** | 岩 + 任意 | 生成护盾，吸收伤害 | 防御 | 保命 |

## 详细机制说明

### 光耀（光 + 雷）
- **触发方式**：光元素攻击命中带有雷元素的敌人，或反之
- **效果**：造成一次范围雷元素伤害，并对周围小范围内其他带雷元素的敌人连锁传导
- **技巧**：先由雷元素角色给多个敌人挂上雷标记，再用光元素角色触发连锁
- **推荐搭配**：元流之子 + 东方曜

### 燃烧（草 + 火）
- **触发方式**：火元素攻击命中带草元素的敌人
- **效果**：敌人进入燃烧状态，每秒受到火元素伤害，持续8秒
- **技巧**：燃烧可叠加层数，最多3层，适合持久战
- **推荐搭配**：蒙犽 + 西施

### 蒸发（火 + 水）
- **触发方式**：火元素攻击命中带水元素的敌人
- **效果**：本次攻击伤害提升100%（2.0倍）
- **技巧**：先挂水再打火伤害最高；如果先挂火再打水，倍率为1.5×
- **推荐搭配**：蒙犽 + 鲁班大师

### 冻结（水 + 冰）
- **触发方式**：冰元素攻击命中带水元素的敌人
- **效果**：敌人被冻结，无法行动3秒
- **技巧**：冻结结束后敌人会短暂「潮湿」，可再次冻结
- **推荐搭配**：冷春 + 鲁班大师 / 王昭君 + 鲁班大师

### 超导（雷 + 冰）
- **触发方式**：雷元素攻击命中带冰元素的敌人
- **效果**：造成范围伤害，并降低受影响敌人30%防御，持续8秒
- **技巧**：超导的减防效果对全队有效，是辅助型反应
- **推荐搭配**：东方曜 + 冷春 / 东方曜 + 王昭君

### 绽放（草 + 水）
- **触发方式**：水元素攻击命中带草元素的敌人
- **效果**：在目标位置生成一片治疗草地，持续恢复站在其中的角色生命
- **技巧**：治疗量与触发者的元素精通有关
- **推荐搭配**：西施 + 鲁班大师 / 孙膑 + 鲁班大师

### 结晶（岩 + 任意）
- **触发方式**：岩元素攻击命中带任意元素的敌人
- **效果**：掉落对应元素的结晶碎片，拾取后获得元素护盾
- **技巧**：护盾强度与铠的生命值上限相关
- **推荐搭配**：铠 + 任意元素输出角色

## 实战配队思路

### 经典「蒸发爆发队」
- **主C**：蒙犽（火）
- **挂水**：鲁班大师（水）
- **增伤**：西施（草，提供元素精通Buff）
- **生存**：铠（岩，结晶护盾）
- **循环**：鲁班大师挂水 → 蒙犽蒸发爆发 → 铠补盾 → 循环

### 经典「感电控制队」
- **主C**：东方曜（雷）
- **副C**：元流之子（光）
- **控制**：王昭君（冰，冻结）
- **治疗**：孙膑（草，绽放治疗）
- **循环**：东方曜挂雷 → 元流之子光耀群攻 → 王昭君冻结残血 → 孙膑治疗

### 经典「超导破甲队」
- **主C**：花木兰（火）
- **副C**：东方曜（雷）
- **减防**：冷春（冰，触发超导）
- **辅助**：伽罗（光，增伤）
- **循环**：冷春挂冰 → 东方曜超导减防 → 花木兰爆发 → 伽罗补刀

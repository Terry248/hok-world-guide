# 全Boss打法与阵容推荐

## 序章Boss

### 秘禁守卫（等级8）
- **位置**：秘禁之地入口
- **弱点**：草元素
- **机制**：
  - 第一阶段：近战挥砍，有较长的前摇
  - 第二阶段（70%以下）：召唤两个石灵小弟
  - 第三阶段（30%以下）：狂暴，攻击速度提升50%
- **推荐阵容**：元流之子（主C）+ 东方曜（助战）+ 西施（挂草）
- **技巧**：第三阶段保持距离，利用东方曜的远程战技消耗

## 世界Boss

### 混沌守卫（等级40）
- **位置**：秘禁之地深处（周常刷新）
- **弱点**：光元素
- **机制**：
  - 拥有护盾，需要先破除护盾才能造成有效伤害
  - 周期性释放全屏AOE，需要躲在场景中的光柱内
  - 掉落高阶突破材料（各元素通用）
- **推荐阵容**：元流之子（破盾）+ 东方曜（输出）+ 孙膑（治疗）+ 铠（抗伤）
- **技巧**：携带至少1个光元素角色破除护盾效率最高

### 山岩巨人（等级35）
- **位置**：观星群山深处
- **弱点**：水元素
- **机制**：
  - 高物理防御，需要用元素反应破防
  - 会投掷巨石，地面有红圈预警
  - 残血时会钻地回血，需要用冰元素冻结打断
- **推荐阵容**：鲁班大师（挂水）+ 蒙犽（蒸发输出）+ 冷春（冰，打断回血）

### 风暴之鹰（等级45）
- **位置**：云落远山高空平台
- **弱点**：冰元素
- **机制**：
  - 飞行单位，近战角色难以命中
  - 会召唤风暴龙卷，需要及时躲避
  - 落地时有较长硬直，是主要输出窗口
- **推荐阵容**：冷春（冰弓，远程输出）+ 王昭君（法杖，控制）+ 蒙犽（火弓，远程蒸发）

## 通用战斗技巧

1. **闪避无敌帧**：闪避动作中有约0.3秒无敌时间，可以躲避任何攻击
2. **元素克制**：攻击被克制元素的敌人时，伤害提升30%
3. **韧性机制**：连续攻击会削减敌人韧性，韧性归零后敌人进入眩晕
4. **空中连击**：部分角色可以将敌人击飞，在空中进行额外攻击
5. **环境互动**：利用场景中的爆炸桶、落石等环境要素辅助战斗

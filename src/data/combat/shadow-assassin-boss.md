# 暗影刺客 Boss 打法详解

暗影刺客是第一章的最终Boss，拥有高机动性和爆发伤害，是新手玩家的第一个挑战。

## Boss 信息

| 属性 | 数值 |
|------|------|
| 等级 | 8 |
| 生命值 | 45,000 |
| 弱点 | 雷元素 |
| 抗性 | 暗元素（免疫50%） |
| 掉落 | 暗影之心 × 3，摩拉 × 20,000 |

## 阶段分析

### 第一阶段（100% - 70% HP）

- **技能**：暗影突袭、暗影之刃
- **行为**：在场地周围高速移动，偶尔现身攻击
- **策略**：使用雷元素角色追踪输出，注意躲避突袭的红圈提示

### 第二阶段（70% - 30% HP）

- **新增技能**：暗影领域（范围AOE）
- **行为**：在场地中心制造暗影领域，持续造成持续伤害
- **策略**：远离暗影领域，等待Boss现身后集中输出

### 第三阶段（30% - 0% HP）

- **新增技能**：暗影分身
- **行为**：制造两个分身，需要找出真身
- **策略**：
  1. 真身的脚下有微弱的光效
  2. 使用AOE技能同时攻击所有分身
  3. 击杀真身后分身自动消失

## 推荐阵容

```
赵云（主C，雷元素）—— 利用感电反应造成持续伤害
貂蝉（副C，风元素）—— 控制Boss走位
蔡文姬（治疗）—— 保证生存
诸葛亮（辅助）—— 提升全队元素精通
```

## 注意事项

- Boss的暗影突袭有0.5秒的前摇，注意观察闪避时机
- 第三阶段不要浪费技能在分身上，找准真身是关键
- 建议角色等级达到8级以上再挑战

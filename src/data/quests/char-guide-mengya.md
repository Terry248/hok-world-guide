# 蒙犽培养攻略

蒙犽是星之队的「火力担当」，性格热血冲动，对火力有着极致的追求。作为火元素弓箭手，他是游戏中少有的远程持续输出角色，独特的「热度」机制让他在长时间战斗中越战越勇。

## 基础信息

| 属性 | 数值 | 评价 |
|------|------|------|
| 生命值 | 9500 | **低** |
| 攻击力 | 370 | **高** |
| 防御力 | 140 | **低** |
| 速度 | 320 | 较快 |
| 暴击率 | 12% | 较高 |
| 暴击伤害 | 60% | 较高 |

**角色定位**：火元素远程持续输出
**武器类型**：弓
**元素属性**：火

## 技能解析

### 普攻：速射弹

使用速射弹模式进行快速射击，命中积攒热度值和兴奋值。可切换至榴弹模式消耗热度值造成范围火元素伤害。

**技巧**：这是蒙犽的核心机制。速射模式单体伤害高，适合打BOSS；榴弹模式范围伤害，适合清小怪。根据战况灵活切换。

### 战技：火力覆盖

进入强化状态，攻击速度大幅提升，每次攻击额外发射追踪导弹。热度值越高，导弹数量和伤害越高。

**技巧**：先普攻攒热度，热度满后再开战技，最大化伤害输出。战技期间尽量保持攻击不要中断。

### 大招：火力全开

架起重型火炮，进入无法移动的重炮模式，对目标区域进行持续轰炸。轰炸期间免疫控制但无法闪避。

**技巧**：大招是双刃剑，输出极高但无法移动。建议在安全位置（如队友保护下或BOSS硬直期）释放。

## 培养优先级

1. **战技等级**：影响攻速提升幅度和导弹伤害，优先升满
2. **普攻等级**：影响热度积攒速度和基础伤害
3. **大招等级**：最后升级，使用频率较低
4. **暴击率/暴击伤害**：蒙犽基础暴击较高，优先堆到70%暴击率后全力堆暴伤

## 武器推荐

| 优先级 | 武器 | 说明 |
|--------|------|------|
| S | 穿云弓（4★） | 攻速提升+命中弱点额外伤害，完美契合蒙犽的快攻速特性 |
| S | 破魔长弓（5★） | 高白值，对元素反应敌人增伤 |
| A | 猎手短弓（3★） | 过渡使用，暴击率副属性 |

**推荐理由**：蒙犽的攻击模式是高频低伤，攻速提升的收益非常高；暴击率副属性能快速达到暴击阈值。

## 配队建议

### 最佳配队：星之队完全体

- **东方曜**（雷 / 主C/近战）
- **蒙犽**（火 / 远程输出）
- **西施**（草 / 治疗）
- **鲁班大师**（雷 / 护盾）

**元素反应**：火+雷触发「超载」造成范围爆炸；草+火触发「燃烧」持续灼烧。
**团队作用**：蒙犽在安全距离持续输出，东方曜近战爆发，远近结合覆盖面广。

### 火核燃烧队

- **蒙犽**（火 / 主C）
- **花木兰**（火 / 副C）
- **西施**（草 / 辅助）
- **鲁班大师**（雷 / 护盾）

**核心思路**：双火核心+草元素辅助，最大化燃烧反应的持续伤害。

## 实战技巧

1. **热度管理**：保持热度值不要归零，热度越高输出越高。切换场景前先普攻把热度攒满
2. **形态切换**：单体敌人用速射，3个以上敌人用榴弹
3. **大招站位**：开大前确保背后安全，最好有队友保护或障碍物遮挡
4. **走位要求**：蒙犽是全游戏最脆的角色之一，必须保持与敌人的距离

## 总结

蒙犽是远程输出的代表角色，370的基础攻击力和12%的基础暴击率在远程角色中非常出色。热度机制让他越战越强，适合持久战。缺点是身板极脆，需要良好的走位意识和队友保护。如果你喜欢远程风筝的玩法，蒙犽是最佳选择。

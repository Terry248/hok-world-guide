# 日常与资源获取完全指南

## 每日必做清单（约30分钟）

| 优先级 | 内容 | 收益 |
|--------|------|------|
| ★★★ | 每日委托 × 4 | 经验、古币、声望 |
| ★★★ | 元素试炼 | 突破材料、武器素材 |
| ★★☆ | 地图探索（标记未收集点） | 宝箱、收集品 |
| ★★☆ | 野外Boss讨伐 | 角色突破材料 |
| ★☆☆ | 食材采集与烹饪 | 战斗Buff道具 |

## 资源获取途径

### 古币
- **主要来源**：每日委托、宝箱开启、任务奖励
- **次要来源**：出售多余材料、完成成就
- **使用优先级**：角色突破 > 武器强化 > 杂货购买

### 突破材料
- **角色突破素材**：对应元素属性的野外Boss掉落
  - 光元素 → 丰云野「光之核」
  - 雷元素 → 观星群山「雷晶蝶」
  - 火元素 → 春溪原「火焰花」
- **武器素材**：元素试炼副本，按日期轮换

### 装备经验
- **主要来源**：分解多余装备、元素试炼
- **次要来源**：大世界采矿（各区域矿点每日刷新）

## 每周必做

- **周常Boss**：秘禁之地深处的「混沌守卫」，掉落高阶突破材料
- **星之队挑战**：组队完成特殊关卡，奖励限定装备
- **稷下学院考核**：周期性开放，奖励大量声望和稀有道具

## 探索效率技巧

1. **传送点优先**：每到一个新区域，先解锁所有传送点
2. **标记系统**：发现暂时无法获取的宝箱时，在地图上标记
3. **元素视野**：使用元素感应可以发现隐藏的可互动物品
4. **昼夜交替**：部分收集品和NPC只在特定时间出现

## 新手常见误区

- ❌ 把古币全部用来抽装备 → ✅ 优先保证角色和武器突破
- ❌ 平均培养所有角色 → ✅ 集中资源培养1主C + 1辅助
- ❌ 忽略日常委托 → ✅ 每日委托是经验和古币的最稳定来源

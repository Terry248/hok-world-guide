# 鲁班大师培养攻略

鲁班大师是稷下学院著名的「神匠」，机关术造诣极深，是星之队的技术担当。作为雷元素辅助，他的核心价值在于为全队提供护盾保护和机关增益。

## 基础信息

| 属性 | 数值 | 评价 |
|------|------|------|
| 生命值 | 14000 | **高** |
| 攻击力 | 240 | 低 |
| 防御力 | 280 | **高** |
| 速度 | 270 | 较慢 |
| 暴击率 | 5% | 低 |
| 暴击伤害 | 50% | 基础 |

**角色定位**：雷元素辅助 / 护盾 / 团队保护
**武器类型**：锤
**元素属性**：雷

## 技能解析

### 普攻：机关连击

挥舞机关锤进行攻击，技能施法或命中目标时叠加护盾（可叠4层），增强生存并增加物攻。

**技巧**：普攻不仅是输出手段，更是叠护盾的途径。保持攻击节奏，让护盾层数始终维持在高位。

### 战技：鲁班5号

召唤机关傀儡「鲁班5号」，持续为附近友方提供护盾，并嘲讽周围敌人攻击傀儡。傀儡存在期间，全队防御力提升。

**技巧**：核心辅助技能，CD好了就放。放在队伍前方吸收伤害，保护脆皮输出角色。

### 大招：机关全开

展开大型机关装置，为全队提供巨额护盾，同时提升全队攻击力和元素精通。护盾存在期间，受到的伤害部分转化为能量回复。

**技巧**：团队保护神技，在BOSS释放高伤害技能前预判释放，或者队友血量危险时紧急使用。

## 培养优先级

1. **大招等级**：影响护盾厚度和增益幅度，优先升满
2. **战技等级**：影响傀儡护盾量和嘲讽范围
3. **普攻等级**：最后升级
4. **生命值/防御力**：护盾量与生命值挂钩，堆血量就是堆盾量

## 武器推荐

| 优先级 | 武器 | 说明 |
|--------|------|------|
| S | 巨岩战锤（5★） | 生命值%+防御力%，最大化护盾量 |
| A | 铁壁之锤（4★） | 防御力%+护盾强效，专门强化护盾 |
| B | 新手战锤（2★） | 过渡使用 |

**推荐理由**：鲁班大师的护盾量基于生命值计算，生命值和防御力是最优先属性。

## 配队建议

### 最佳配队：星之队完全体

- **东方曜**（雷 / 主C）
- **蒙犽**（火 / 远程输出）
- **西施**（草 / 治疗）
- **鲁班大师**（雷 / 护盾）

**团队作用**：鲁班大师提供护盾+嘲讽，西施提供治疗，双重保护让脆皮输出角色可以安心输出。

### 铁壁生存队

- **铠**（冰 / 主C）
- **鲁班大师**（雷 / 护盾）
- **西施**（草 / 治疗）
- **王昭君**（冰 / 控制）

**特点**：双护盾+治疗+控制，生存能力拉满，适合磨血型BOSS战。

## 实战技巧

1. **护盾覆盖**：始终保持护盾在队友身上，不要等盾破了再补
2. **傀儡位置**：把鲁班5号放在敌人密集区域，最大化嘲讽效果
3. **大招预判**：观察BOSS攻击前摇，提前开大保护全队
4. **走位要求**：虽然自己肉，但也要避免被集火，否则护盾断档很危险

## 总结

鲁班大师是游戏中最可靠的护盾辅助，14000的生命值和280的防御力让他成为团队的定海神针。在高压环境下（如高难副本、限时挑战），有鲁班大师的保护和没有完全是两种体验。推荐给所有追求稳定通关的玩家。

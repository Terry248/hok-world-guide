# 元流之子培养攻略

元流之子是《王者荣耀世界》的主角，失去过往记忆的稷下新生，也是玩家的化身。作为光元素输出角色，他拥有独特的「共鸣」系统，可与其他英雄建立联系，切换不同的战技形态。

## 基础信息

| 属性 | 数值 | 评价 |
|------|------|------|
| 生命值 | 12000 | 中等 |
| 攻击力 | 320 | 中等 |
| 防御力 | 200 | 中等 |
| 速度 | 300 | 中等 |
| 暴击率 | 5% | 较低 |
| 暴击伤害 | 50% | 基础 |

**角色定位**：光元素主C / 可切换形态的多功能输出
**武器类型**：剑
**元素属性**：光

## 技能解析

### 普攻：万物为流

连续攻击汇集「流」能量，满5层后减少50%其他技能冷却，并回复大量生命值。这是元流之子最核心的续航机制，实战中需要保持攻击节奏，尽快叠满5层流。

**技巧**：面对群怪时优先普攻攒流，满层后再释放战技和大招，形成技能循环。

### 战技：倾注一击

释放能量波造成多段光元素伤害。根据当前共鸣英雄的不同，战技会切换为不同形态：

- **流转轮形态**：范围更广，适合清杂
- **流冲枪形态**：单体爆发更高

**技巧**：根据敌人数量和类型灵活切换共鸣形态。打BOSS切单体形态，清小怪切范围形态。

### 大招：流脉觉醒

汇聚光元素之力，释放强力范围攻击。大招期间获得霸体效果，免疫控制。

**技巧**：大招霸体可用于硬抗BOSS高伤害技能，是保命+输出的双重手段。

## 培养优先级

1. **普通攻击等级**：优先升满，因为普攻是攒流的核心手段
2. **战技等级**：次优先，影响主要输出
3. **大招等级**：最后升级，虽然强力但CD较长
4. **暴击率/暴击伤害**：元流之子基础暴击较低，需要通过装备补足

## 武器推荐

| 优先级 | 武器 | 说明 |
|--------|------|------|
| S | 辉星剑（4★） | 副属性攻击力%，被动光元素溅射，与光元素完美契合 |
| A | 破魔长剑（5★） | 高白值，对元素反应状态敌人增伤，泛用性强 |
| B | 寒星剑（2★） | 新手过渡，攻速快适合快速攒流 |

**推荐理由**：元流之子基础攻击面板中等，需要攻击力%副属性弥补；光元素溅射能提升清杂效率。

## 配队建议

### 最佳配队：光雷双核

- **元流之子**（光 / 主C）
- **东方曜**（雷 / 副C）
- **西施**（草 / 辅助/治疗）
- **鲁班大师**（雷 / 护盾辅助）

**元素共鸣**：光+雷触发「光耀雷鸣」，提升全队攻击速度和元素充能效率。

### 新手配队：稷下学院队

- **元流之子**（光 / 主C）
- **东方曜**（雷 / 副C）
- **西施**（草 / 辅助）
- **孙膑**（风 / 加速辅助）

**羁绊加成**：4人同出「稷下学院」，获得元素精通+15%的团队增益。

## 实战技巧

1. **保持攻击节奏**：元流之子的强度取决于流的层数管理，不要打断普攻连招
2. **共鸣切换时机**：进入战斗前根据敌人类型切换共鸣形态，战斗中切换有CD限制
3. **大招保命**：血量危险时直接开大招，霸体+高回复能救你一命
4. **走位要点**：元流之子攻击范围中等，需要贴近敌人但不要贪刀

## 总结

元流之子是一个上限很高的主角角色，共鸣系统带来了极高的可玩性。虽然基础面板不出彩，但通过合理的装备搭配和配队，可以成为队伍的核心输出。新手建议优先投入资源培养，因为主线剧情中强制出场率很高。

# 东方曜培养攻略

东方曜，自称"星之队"队长，是稷下学院中极具个性的学生。作为雷元素输出角色，他拥有游戏中最高的爆发伤害能力之一，是追求极限输出玩家的首选。

## 基础信息

| 属性 | 数值 | 评价 |
|------|------|------|
| 生命值 | 11500 | 中等偏低 |
| 攻击力 | 380 | **极高** |
| 防御力 | 180 | 较低 |
| 速度 | 330 | **快** |
| 暴击率 | 10% | 中等 |
| 暴击伤害 | 55% | 中等 |

**角色定位**：雷元素爆发主C
**武器类型**：剑
**元素属性**：雷

## 技能解析

### 普攻：星辰之剑

四段连续斩击，攻击范围广，清杂能力强。蓄力普攻可向前突进并发动强力挥砍。

**技巧**：常规输出使用普通四连击，需要快速接近敌人时使用蓄力突进。

### 战技：星辰裂斩

释放雷元素剑气，对前方扇形区域造成多段雷元素伤害。命中敌人后获得「星痕」层数，提升下次大招伤害。

**技巧**：每次大招前先释放战技叠满星痕，确保大招吃到增伤加成。

### 大招：星辰陨落

跃起后俯冲，对目标区域造成巨额雷元素范围伤害。若目标处于元素反应状态，伤害进一步提升。

**技巧**：大招有较长前摇，建议配合控制角色（如王昭君）冻结敌人后再释放，确保命中。

## 培养优先级

1. **大招等级**：东方曜的核心输出手段，优先升满
2. **战技等级**：影响星痕叠层效率和伤害
3. **普攻等级**：最后升级，主要用于填充技能空窗期
4. **暴击伤害**：基础暴击率10%，堆到60-70%后全力堆暴伤

## 武器推荐

| 优先级 | 武器 | 说明 |
|--------|------|------|
| S | 破魔长剑（5★） | 高白值+攻击力%，被动对元素反应敌人增伤，完美契合东方曜的爆发机制 |
| S | 辉星剑（4★） | 攻击力%+光元素溅射，虽然是光元素但溅射伤害能补充AOE |
| A | 寒星剑（2★） | 新手过渡，攻速快适合频繁触发普攻 |

**推荐理由**：东方曜攻击力面板高达380，需要高白值武器进一步放大优势；被动增伤对爆发角色收益极高。

## 配队建议

### 最佳配队：星之队完全体

- **东方曜**（雷 / 主C）
- **西施**（草 / 辅助/治疗）
- **蒙犽**（火 / 副C/远程）
- **鲁班大师**（雷 / 护盾辅助）

**元素共鸣**：雷+火触发「雷霆烈焰」，降低敌人防御力；草+雷触发「寄生雷光」，持续造成雷元素伤害。
**羁绊加成**：4人同属「星之队」，全队暴击率+10%。

### 速切爆发队

- **东方曜**（雷 / 主C）
- **元流之子**（光 / 副C）
- **王昭君**（冰 / 控制）
- **孙膑**（风 / 加速）

**输出循环**：王昭君冻结 → 东方曜大招 → 元流之子战技 → 孙膑加速 → 下一轮

## 实战技巧

1. **星痕管理**：每次大招前必须先用战技叠层，这是东方曜输出的核心
2. **走位意识**：防御力只有180，被BOSS打中很疼，需要灵活走位
3. **技能连招**：战技 → 普攻攒能量 → 大招，这是最基本的循环
4. **元素反应配合**：让队友先手挂元素（王昭君挂冰、西施挂草），东方曜再触发雷元素反应

## 总结

东方曜是游戏中最纯粹的爆发输出角色，380的基础攻击力和330的速度让他在短时间内能打出极高的伤害。缺点是身板较脆，需要护盾或治疗角色保护。如果你喜欢大数字秒怪的快感，东方曜绝对值得优先培养。

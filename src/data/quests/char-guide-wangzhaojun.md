# 王昭君培养攻略

王昭君是来自北荒的「冰霜掌控者」，外表冷艳，拥有操控冰雪的强大力量。作为冰元素控制角色，她是游戏中最强的控场法师，擅长通过冻结效果限制敌人行动。

## 基础信息

| 属性 | 数值 | 评价 |
|------|------|------|
| 生命值 | 10000 | 偏低 |
| 攻击力 | 310 | 中等 |
| 防御力 | 150 | 低 |
| 速度 | 300 | 中等 |
| 暴击率 | 8% | 较低 |
| 暴击伤害 | 55% | 中等 |

**角色定位**：冰元素控制 / 范围输出
**武器类型**：法杖
**元素属性**：冰

## 技能解析

### 普攻：冰霜法球

释放冰霜法球攻击敌人，造成冰元素伤害并附带减速效果。连续命中同一目标会叠加「霜冻」层数。

**技巧**：普攻是叠加霜冻的主要手段，保持对同一目标的攻击节奏，尽快叠满层数触发冻结。

### 战技：冰晶扇击

挥舞冰晶扇释放扇形冰元素冲击，对前方敌人造成冰元素伤害并大幅叠加霜冻层数。被命中的敌人移动速度大幅降低。

**技巧**：范围控场技能，面对群怪时效果极佳。配合大招可以快速冻结多个敌人。

### 大招：绝对零度

在指定区域召唤暴风雪，持续造成冰元素范围伤害并快速叠加霜冻层数。暴风雪结束时，区域内所有敌人被冻结。

**技巧**：最强控场技能，打BOSS时用于打断关键技能，清小怪时可以冻结全场让队友收割。

## 培养优先级

1. **大招等级**：影响暴风雪伤害和冻结效率，优先升满
2. **战技等级**：影响霜冻叠加速度和减速幅度
3. **普攻等级**：影响基础伤害和霜冻叠加
4. **元素精通**：冰元素反应伤害与元素精通挂钩

## 武器推荐

| 优先级 | 武器 | 说明 |
|--------|------|------|
| S | 霜华法杖（5★） | 元素精通+冰元素伤害加成，最大化控制和反应伤害 |
| A | 星辉法杖（4★） | 元素精通副属性，提升冰元素反应伤害 |
| B | 新手法杖（2★） | 过渡使用 |

**推荐理由**：王昭君的核心价值在控制，元素精通能提升冰元素反应的伤害和效果。

## 配队建议

### 最佳配队：融化爆发队

- **东方曜**（雷 / 主C）
- **王昭君**（冰 / 控制）
- **花木兰**（火 / 副C）
- **西施**（草 / 治疗）

**核心思路**：王昭君冻结 → 花木兰火元素触发「融化」造成巨额伤害 → 东方曜补刀。控制链+爆发伤害的组合。

### 冰控永冻队

- **王昭君**（冰 / 控制）
- **铠**（冰 / 坦克）
- **元流之子**（光 / 主C）
- **西施**（草 / 治疗）

**特点**：双冰快速叠加霜冻，几乎可以让敌人一直处于冻结或减速状态。

## 实战技巧

1. **冻结优先级**：优先冻结高威胁敌人（精英怪、BOSS），不要浪费在小怪身上
2. **大招时机**：BOSS读条大招时立即开大冻结打断，这是王昭君最核心的价值
3. **走位意识**：身板很脆，必须保持在队伍后方，不要冲到前线
4. **元素附着**：先让队友挂其他元素（火/雷），王昭君再挂冰触发反应

## 总结

王昭君是游戏中最强的控场角色，虽然没有高额爆发，但冻结效果能为全队创造绝佳的输出环境。在高难副本和BOSS战中，王昭君的控制能力往往是通关的关键。如果你喜欢掌控战局、为队友创造机会的玩法，王昭君是不二之选。

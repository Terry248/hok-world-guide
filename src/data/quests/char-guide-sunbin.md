# 孙膑培养攻略

孙膑是星之队的「时光守护者」，性格开朗乐观，对机械和时间魔法有着独特的天赋。作为风元素辅助，他的核心价值在于团队加速和控场能力。

## 基础信息

| 属性 | 数值 | 评价 |
|------|------|------|
| 生命值 | 10200 | 偏低 |
| 攻击力 | 250 | 较低 |
| 防御力 | 150 | 低 |
| 速度 | 320 | 较快 |
| 暴击率 | 5% | 低 |
| 暴击伤害 | 50% | 基础 |

**角色定位**：风元素辅助 / 团队加速 / 控场
**武器类型**：法杖
**元素属性**：风

## 技能解析

### 普攻：时空打击

操控时间之力进行攻击。第四次普攻附带轻微击退效果。被动效果：释放技能后增加自身移速。

**技巧**：普攻伤害较低，主要用于填充技能CD和积攒能量。注意第四次普攻的击退可以打断小怪动作。

### 战技：时光回溯

为指定友方角色施加时光印记，短时间内受到伤害的一部分会在印记结束时转化为治疗回复。同时提升该角色的攻击速度。

**技巧**：核心辅助技能，给主C（如东方曜）使用效果最好。预判BOSS高伤害技能前释放，能抵消大量伤害。

### 大招：时光领域

展开大范围时间领域，领域内所有友方角色移速和攻速大幅提升，敌方角色移速降低。领域结束时对范围内敌人造成风元素伤害。

**技巧**：团队增益神技，打BOSS时开场就放，让全队输出节奏快人一步。配合高爆发角色收益最大。

## 培养优先级

1. **大招等级**：影响加速幅度和领域伤害，优先升满
2. **战技等级**：影响伤害转化比例和攻速提升幅度
3. **普攻等级**：最后升级
4. **元素充能效率**：孙膑大招能量需求较高，需要堆充能效率保证大招覆盖率

## 武器推荐

| 优先级 | 武器 | 说明 |
|--------|------|------|
| S | 流风法杖（4★/5★） | 元素充能效率副属性，保证大招循环 |
| A | 星辉法杖（4★） | 元素精通副属性，提升风元素扩散伤害 |
| B | 新手法杖（2★） | 过渡使用 |

**推荐理由**：孙膑的核心价值在大招的团队加速，需要保证大招不断档，因此元素充能效率是最优先属性。

## 配队建议

### 最佳配队：速切爆发队

- **东方曜**（雷 / 主C）
- **元流之子**（光 / 副C）
- **孙膑**（风 / 加速辅助）
- **西施**（草 / 治疗）

**核心思路**：孙膑大招加速 → 东方曜和元流之子快速释放技能 → 孙膑战技保护 → 下一轮循环。攻速提升让爆发角色的技能前摇更短，输出更流畅。

### 新手友好队

- **元流之子**（光 / 主C）
- **孙膑**（风 / 辅助）
- **西施**（草 / 治疗）
- **鲁班大师**（雷 / 护盾）

**特点**：生存能力极强，适合不熟悉走位的玩家。

## 实战技巧

1. **大招时机**：打BOSS开场就放，不要等；清小怪时可以等敌人聚集后再放
2. **战技目标选择**：优先给主C或即将受到高伤害的队友
3. **走位要求**：孙膑防御很低，必须保持在队伍最后方
4. **风元素扩散**：大招的风元素可以扩散其他元素，配合元素反应队效果很好

## 总结

孙膑是一个团队增益型辅助，个人输出能力很弱，但能让整个团队的输出效率提升一个档次。他的加速效果在限时挑战和高难副本中尤为重要。如果你的队伍缺速度、缺灵活性，孙膑是最好的选择。

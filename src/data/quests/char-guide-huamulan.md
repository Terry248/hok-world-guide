# 花木兰培养攻略

花木兰是「长城守卫军」的核心战力，性格坚毅果敢，有着不输男儿的武艺。作为火元素输出，她独特的轻剑/重剑双形态让她在灵活与爆发之间自由切换。

## 基础信息

| 属性 | 数值 | 评价 |
|------|------|------|
| 生命值 | 13000 | 较高 |
| 攻击力 | 360 | 较高 |
| 防御力 | 220 | 较高 |
| 速度 | 320 | 较快 |
| 暴击率 | 12% | 较高 |
| 暴击伤害 | 60% | 较高 |

**角色定位**：火元素输出 / 双形态战士
**武器类型**：剑
**元素属性**：火

## 技能解析

### 普攻：轻剑连斩

挥舞轻剑进行最多五段连续斩击，叠加5层锋痕触发沉默+减速效果。蓄力普攻可向前突进并发动强力挥砍。

**技巧**：轻剑模式攻速快，适合快速叠层和追击敌人。5层锋痕的沉默效果可以打断BOSS技能。

### 战技：形态切换

在轻剑和重剑形态之间切换。轻剑形态攻速快、机动性高；重剑形态攻击力大幅提升，但攻速降低。切换时获得短暂无敌帧。

**技巧**：切换时的无敌帧可以用来躲避BOSS高伤害技能，这是花木兰的核心操作技巧。

### 大招：绽放之刃

**轻剑模式**：快速突进并连续斩击，造成多段火元素伤害。
**重剑模式**：蓄力后释放强力下劈，造成巨额单体火元素伤害并击退敌人。

**技巧**：根据敌人数量选择模式。单体BOSS用重剑大招，群怪用轻剑大招。

## 培养优先级

1. **大招等级**：两种形态大招都受益，优先升满
2. **战技等级**：影响切换后的属性加成幅度
3. **普攻等级**：影响轻剑叠层效率
4. **暴击率/暴击伤害**：基础暴击较高，优先堆到70%暴击率

## 武器推荐

| 优先级 | 武器 | 说明 |
|--------|------|------|
| S | 破魔长剑（5★） | 高白值+攻击力%，对元素反应敌人增伤，双形态都受益 |
| A | 辉星剑（4★） | 攻击力%+元素溅射，轻剑模式清杂效率更高 |
| B | 寒星剑（2★） | 过渡使用，轻剑模式下攻速收益更高 |

**推荐理由**：花木兰两种形态都需要攻击力支撑，因此攻击力%是最优先属性。5星武器的高白值对重剑模式收益更大。

## 配队建议

### 最佳配队：长城守卫军

- **花木兰**（火 / 主C）
- **铠**（冰 / 副C）
- **西施**（草 / 治疗）
- **鲁班大师**（雷 / 护盾）

**羁绊加成**：2人同属「长城守卫军」，获得攻击力+10%。
**元素反应**：冰+火触发「融化」造成额外伤害；火+雷触发「超载」范围爆炸。

### 火核爆发队

- **花木兰**（火 / 主C）
- **蒙犽**（火 / 远程输出）
- **西施**（草 / 辅助）
- **孙膑**（风 / 加速）

**核心思路**：双火核心+风元素扩散，最大化火元素覆盖和反应伤害。

## 实战技巧

1. **形态切换节奏**：轻剑攒层 → 切重剑爆发 → 切轻剑追击，形成循环
2. **无敌帧利用**：切换形态的无敌帧可以躲避任何伤害，多练习时机把握
3. **沉默打断**：轻剑5层锋痕可以沉默敌人，在BOSS读条技能时使用
4. **重剑站位**：重剑攻速慢，确保命中再出手，不要空挥

## 总结

花木兰是一个操作上限很高的角色，双形态带来了极高的可玩性。13000的生命值和220的防御力让她比纯输出角色更耐打，12%的基础暴击率也降低了装备成型难度。如果你喜欢有一定操作深度的角色，花木兰是很好的选择。

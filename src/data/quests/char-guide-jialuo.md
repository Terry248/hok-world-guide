# 伽罗培养攻略

伽罗是来自云中漠地千窟城的「破魔之箭」，精通弓箭之术，性格冷静沉稳。作为风元素远程输出，她拥有游戏中最强的破盾能力，是面对高防御敌人的首选角色。

## 基础信息

| 属性 | 数值 | 评价 |
|------|------|------|
| 生命值 | 9200 | **最低** |
| 攻击力 | **390** | **最高** |
| 防御力 | 130 | **最低** |
| 速度 | **340** | **最快** |
| 暴击率 | **15%** | **最高** |
| 暴击伤害 | **65%** | **最高** |

**角色定位**：风元素远程输出 / 破盾专家
**武器类型**：弓
**元素属性**：风

## 技能解析

### 普攻：破魔之箭

连续射击，长弓/短弓双形态切换。长弓状态下普攻间隔随攻击次数递减，攻速不断提升。所有普攻和技能伤害优先对目标护盾造成等额伤害，破盾能力极强。

**技巧**：核心机制——破盾。面对带盾敌人时，伽罗的伤害优先破盾，破完盾后才对本体造成伤害。长弓模式适合打BOSS，短弓模式适合清小怪。

### 战技：风之矢

释放风元素箭矢，穿透路径上所有敌人并附加「破魔」标记。被标记的敌人护盾值额外降低，且受到的下一次元素反应伤害提升。

**技巧**：面对带盾精英怪时先放战技挂标记，然后普攻快速破盾。标记也能让队友的下一次反应伤害更高。

### 大招：风魔乱舞

进入高速射击状态，攻速大幅提升，每次射击附加风元素伤害并击退敌人。射击会自动追踪被破魔标记的敌人。

**技巧**：爆发输出技能，配合战技标记使用效果最好。击退效果可以自保，防止敌人近身。

## 培养优先级

1. **普攻等级**：影响基础伤害和破盾效率，优先升满
2. **大招等级**：影响爆发期间的伤害和击退效果
3. **战技等级**：影响标记的持续时间和破盾加成
4. **暴击率/暴击伤害**：基础暴击已经很高，优先堆到80%暴击率后全力堆暴伤

## 武器推荐

| 优先级 | 武器 | 说明 |
|--------|------|------|
| S | 穿云弓（4★） | 攻速提升+命中弱点额外伤害，完美契合伽罗的快攻速特性 |
| S | 破魔长弓（5★） | 高白值，对元素反应敌人增伤，泛用性强 |
| A | 猎手短弓（3★） | 过渡使用 |

**推荐理由**：伽罗基础暴击率15%已经很高，穿云弓的攻速提升能让破盾效率更高。破魔长弓的高白值能放大390基础攻击力的优势。

## 配队建议

### 最佳配队：破盾速杀队

- **伽罗**（风 / 远程输出/破盾）
- **东方曜**（雷 / 主C）
- **西施**（草 / 治疗）
- **鲁班大师**（雷 / 护盾）

**核心思路**：伽罗快速破盾 → 东方曜近战爆发输出。风+雷触发「扩散」传播元素，提升范围伤害。

### 风筝队

- **伽罗**（风 / 远程输出）
- **蒙犽**（火 / 远程输出）
- **孙膑**（风 / 加速）
- **西施**（草 / 治疗）

**特点**：双远程+加速+治疗，全程风筝敌人，几乎不会受到近战伤害。

## 实战技巧

1. **破盾优先**：遇到带盾敌人优先攻击，伽罗的破盾速度是全游戏最快的
2. **形态切换**：单体用长弓攒攻速，群怪用短弓快速清场
3. **走位至上**：9200的生命值和130的防御力是全游戏最低，被摸到就是半条命
4. **保持距离**：利用340的速度优势，始终保持与敌人的安全距离

## 总结

伽罗是游戏中最极端的远程输出角色——390的攻击力和15%的暴击率让她输出能力爆表，但9200的生命值和130的防御力也让她脆弱无比。破盾机制让她在特定场合（如盾系BOSS战）有不可替代的价值。如果你对自己的走位有信心，伽罗能给你带来最爽快的远程输出体验。

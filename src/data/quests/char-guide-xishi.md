# 西施培养攻略

西施是稷下学院的「宝藏少女」，性格活泼乐观，是星之队的重要成员。作为草元素辅助，她拥有游戏中最独特的治疗机制——通过反弹治疗效果为队友回复生命。

## 基础信息

| 属性 | 数值 | 评价 |
|------|------|------|
| 生命值 | 10000 | 偏低 |
| 攻击力 | 280 | 较低 |
| 防御力 | 160 | 较低 |
| 速度 | 310 | 中等 |
| 暴击率 | 5% | 低 |
| 暴击伤害 | 50% | 基础 |

**角色定位**：草元素辅助 / 治疗 / 副输出
**武器类型**：法杖
**元素属性**：草

## 技能解析

### 普攻：幻纱之灵

操控幻纱进行攻击，获得「幻纱缠心」状态。进入该状态期间，普攻可以反弹治疗效果给友方，接下来的2次技能施放会推出3个法球且伤害提升。

**技巧**：这是西施最核心的机制。先进入幻纱缠心状态，然后普攻攻击敌人，伤害的一部分会转化为治疗反弹给队友。需要掌握好状态切换时机。

### 战技：幻纱牵引

释放幻纱能量，对前方敌人造成草元素伤害并牵引聚集。被牵引的敌人会被施加「缠心」标记，受到草元素相关反应时额外受到伤害。

**技巧**：聚怪效果在面对分散的小怪时非常实用，配合范围输出角色（如东方曜）效率极高。

### 大招：幻纱领域

展开大范围幻纱领域，持续造成草元素伤害。领域内的友方角色获得持续治疗效果，同时提升草元素反应的伤害。

**技巧**：大招是团队增益型技能，CD好了就放，不要吝啬。在BOSS战中放在队友集中的位置。

## 培养优先级

1. **大招等级**：影响治疗量和领域伤害，优先升满
2. **战技等级**：影响聚怪效率和缠心标记伤害
3. **普攻等级**：影响幻纱缠心状态的输出和治疗转化率
4. **元素精通**：草元素反应伤害和治疗量都与元素精通挂钩

## 武器推荐

| 优先级 | 武器 | 说明 |
|--------|------|------|
| S | 星辉法杖（4★/5★） | 元素精通副属性，提升草元素反应和治疗量 |
| A | 晨曦之杖（4★） | 攻击力%+治疗加成，平衡输出与治疗 |
| B | 新手法杖（2★） | 过渡使用 |

**推荐理由**：西施作为辅助，元素精通是最核心的属性，直接影响草元素反应伤害和治疗转化率。

## 配队建议

### 最佳配队：星之队完全体

- **东方曜**（雷 / 主C）
- **西施**（草 / 辅助/治疗）
- **蒙犽**（火 / 副C）
- **鲁班大师**（雷 / 护盾）

**元素反应**：草+雷触发「寄生雷光」持续伤害；草+火触发「燃烧」持续灼烧。
**团队作用**：西施提供聚怪、治疗和草元素附着，是团队的生存保障。

### 生存保障队

- **铠**（冰 / 主C）
- **西施**（草 / 治疗）
- **鲁班大师**（雷 / 护盾）
- **王昭君**（冰 / 控制）

**特点**：双冰控制+草治疗+雷护盾，生存能力极强，适合高难副本。

## 实战技巧

1. **幻纱缠心循环**：进入状态 → 普攻3次触发治疗反弹 → 释放战技推出法球 → 重新进入状态
2. **走位意识**：西施身板很脆，不要站在前线，保持在队伍后方
3. **大招释放时机**：队友血量低于50%时立即开大，或者BOSS释放全屏技能前预判开大
4. **聚怪配合**：战技牵引后立刻通知队友释放范围技能

## 总结

西施是游戏中最全面的辅助角色之一，集治疗、聚怪、元素附着于一身。虽然个人输出不高，但能大幅提升整个团队的生存能力和输出效率。对于喜欢辅助定位的玩家来说，西施是必练角色。在高难副本中，有西施和没西施完全是两种体验。

# 铠培养攻略

铠是「长城守卫军」的守护者，性格沉稳内敛，背负着沉重的过去。作为冰元素角色，他拥有全游戏最高的防御面板和独特的反击机制，是最可靠的坦克型输出。

## 基础信息

| 属性 | 数值 | 评价 |
|------|------|------|
| 生命值 | **14500** | **最高** |
| 攻击力 | 340 | 中等 |
| 防御力 | **250** | **最高** |
| 速度 | 290 | 较慢 |
| 暴击率 | 8% | 较低 |
| 暴击伤害 | 55% | 中等 |

**角色定位**：冰元素坦克 / 反击输出 / 团队前排
**武器类型**：剑
**元素属性**：冰

## 技能解析

### 普攻：冰甲剑术

挥舞冰霜之剑进行连续攻击，造成冰元素伤害。攻击附带减速效果，降低敌人移动速度。

**技巧**：普攻的减速效果可以限制敌人走位，为队友创造输出环境。攻速虽然不快但每击都很有分量。

### 战技：极寒之刃

释放冰元素能量，为自身附加「冰甲」状态。冰甲状态下防御力大幅提升，且受到攻击时会自动反击，对攻击者造成冰元素伤害并附加减速。

**技巧**：核心技能，打BOSS时保持冰甲不断。反击伤害基于防御力计算，因此堆防御就是堆输出。

### 大招：冰封领域

展开大范围冰封领域，领域内所有敌人持续受到冰元素伤害并叠加「霜冻」层数。层数满时敌人会被冻结，无法行动。

**技巧**：强力控场技能，面对群怪时效果极佳。冻结效果可以打断BOSS技能，为全队创造输出窗口。

## 培养优先级

1. **战技等级**：影响冰甲防御加成和反击伤害，优先升满
2. **大招等级**：影响领域伤害和冻结效率
3. **普攻等级**：最后升级
4. **防御力**：铠的核心属性，防御越高反击越痛

## 武器推荐

| 优先级 | 武器 | 说明 |
|--------|------|------|
| S | 冰封之剑（5★） | 防御力%+冰元素伤害加成，完美契合铠的机制 |
| A | 巨岩战剑（4★） | 生命值%+防御力%，最大化生存能力 |
| B | 寒星剑（2★） | 过渡使用 |

**推荐理由**：铠的伤害主要来自防御力转化的反击，因此防御力是最优先属性。冰元素加成能提升技能伤害。

## 配队建议

### 最佳配队：长城守卫军

- **花木兰**（火 / 主C）
- **铠**（冰 / 坦克/副C）
- **西施**（草 / 治疗）
- **鲁班大师**（雷 / 护盾）

**羁绊加成**：2人同属「长城守卫军」，获得攻击力+10%。
**元素反应**：冰+火触发「融化」额外伤害；冰+雷触发「超导」降低敌人物防。

### 冰控队

- **铠**（冰 / 坦克/控制）
- **王昭君**（冰 / 控制/输出）
- **元流之子**（光 / 主C）
- **西施**（草 / 治疗）

**特点**：双冰控制+光元素输出，控制链非常稳定。

## 实战技巧

1. **冰甲保持**：战技CD好了就续，不要断冰甲
2. **吸引仇恨**：主动贴近敌人，让敌人攻击你触发反击
3. **大招控场**：群怪时先开大冻结，让队友集中输出
4. **防御即输出**：不要嫌弃防御词条，对铠来说防御就是伤害

## 总结

铠是游戏中最硬的坦克角色，14500的生命值和250的防御力让他可以硬抗大多数伤害。独特的反击机制让他在挨打的同时还能输出，非常适合放在队伍前排吸收伤害。推荐给喜欢稳扎稳打、不爱走位的玩家。有铠在，团队生存压力会小很多。

# 腾讯云轻量服务器 - 重置密码步骤

## 方法：在控制台重置密码

### 步骤 1：登录腾讯云控制台
打开：https://console.cloud.tencent.com/lighthouse

### 步骤 2：找到你的服务器
1. 左侧菜单点击 **「轻量应用服务器」**
2. 在服务器列表中找到你的实例（应该显示北京地域）
3. 点击实例名称或 **「管理」** 进入详情页

### 步骤 3：重置密码
在实例详情页：
1. 找到 **「更多操作」** 或右上角 **「...」** 按钮
2. 点击下拉菜单中的 **「重置密码」**
   - 如果没有，可能在 **「操作」** 列里
3. 输入新密码（比如：`Nicholas1111` 或你自己设一个）
4. 点击 **「确认」**
5. 在弹窗中点击 **「同意强制关机」**（重置密码需要重启服务器）

### 步骤 4：等待重启（约1-2分钟）
服务器状态会从 **「运行中」** → **「关机中」** → **「运行中」**

### 步骤 5：开启密码登录（关键！）
重置密码后，还需要确保服务器允许密码登录：

**方式 A：在腾讯云控制台直接修改（推荐）**
1. 在实例详情页，找到 **「远程登录」** 或 **「登录」** 按钮
2. 点击后选择 **「一键登录」**（通过浏览器WebShell登录）
3. 登录后执行：
   ```bash
   sudo sed -i 's/PasswordAuthentication no/PasswordAuthentication yes/' /etc/ssh/sshd_config
   sudo sed -i 's/#PasswordAuthentication yes/PasswordAuthentication yes/' /etc/ssh/sshd_config
   sudo systemctl restart sshd
   ```

**方式 B：如果控制台有「登录方式设置」**
1. 在实例详情页找到 **「登录设置」** 或 **「绑定密钥」**
2. 修改为 **「密码登录」** 模式

---

## 验证连接

重置密码并开启密码登录后，测试：

```bash
ssh ubuntu@2402:4e00:c030:7600:bad6:572c:26a0:0
# 输入你设置的密码
```

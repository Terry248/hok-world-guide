# 腾讯优量汇申请指南

> 适用网站：hokworld.site（国内版）
> 备案号：赣ICP备2026010150号

---

## 一、申请前准备

### 已满足条件
- ✅ 网站已备案（赣ICP备2026010150号）
- ✅ 网站底部已展示备案号
- ✅ 网站内容健康，无违规信息
- ✅ 60+页面，内容充实
- ✅ 有隐私政策、免责声明页面

### 申请账号
1. 访问 https://e.qq.com/ads/
2. 用QQ或微信扫码登录
3. 选择「媒体合作」→「网站」
4. 进入「优量汇」平台

---

## 二、填写资料

### 网站基本信息

| 字段 | 填写内容 |
|------|---------|
| 网站名称 | **王者大陆探索志** |
| 网站域名 | hokworld.site |
| 网站类型 | 游戏资讯/攻略 |
| 网站描述 | 个人游戏数据整理网站，收集整理游戏内的角色信息、武器装备数据和地图探索内容 |
| ICP备案号 | 赣ICP备2026010150号 |
| 日均PV | 如实填写（新站可先预估 500-2000）|
| 网站语言 | 中文 |

### 为什么选择「王者大陆探索志」而不是「王者荣耀世界攻略站」？
- 避免直接使用「王者荣耀」注册商标
- 腾讯优量汇与腾讯游戏生态打通，对游戏类站点审核相对友好
- 「探索志」偏向个人记录，减少商业化嫌疑

## 三、广告位创建（审核通过后）

审核通过后，在优量汇后台：

1. 创建「展示广告」类型
2. 选择「PC网页」或「移动网页」
3. 填写广告位名称（如：侧边栏广告、底部Banner）
4. 选择尺寸：
   - 桌面侧边栏：160×600 或 300×600
   - 移动端底部：320×50 或 300×250

### 广告位ID填入代码

拿到广告位ID后，修改文件：

```ts
// src/lib/ad-config.ts
export const GDT_CONFIG = {
  appId: '你的媒体ID',           // ← 填入
  placements: {
    sidebarLeft: '广告位ID_1',   // ← 填入
    sidebarRight: '广告位ID_2',  // ← 填入
    mobileBanner: '广告位ID_3',  // ← 填入
    inFeed: '广告位ID_4',        // ← 可选
  },
};
```

## 六、常见问题

**Q：游戏类网站能通过审核吗？**
A：腾讯优量汇与腾讯游戏生态打通，游戏资讯/攻略类站点审核通过率较高，但需要内容健康、无违规。

**Q：需要多少流量才能通过？**
A：优量汇对流量要求比百度联盟宽松，新站也有机会通过，如实填写即可。

**Q：审核被拒怎么办？**
A：常见原因：
- 网站内容不足 → 继续添加攻略文章
- 备案信息不一致 → 检查备案号是否正确
- 网站无法访问 → 确认服务器正常运行
- 根据拒绝原因修改后重新提交

# 王者荣耀世界攻略站 - 服务器部署指南

## 服务器信息
- **类型**: 腾讯云轻量应用服务器
- **配置**: 4核4G3M 或 2核4G5M
- **系统**: Ubuntu 22.04 LTS
- **部署路径**: `/var/www/hok-world-guide/`

---

## 第一步：连接服务器

```bash
# 获取服务器公网IP后连接
ssh ubuntu@你的公网IP

# 或如果用root
ssh root@你的公网IP
```

> 腾讯云轻量服务器默认用户是 `ubuntu` 或 `root`，密码在购买时设置/自动生成。

---

## 第二步：运行自动部署脚本

```bash
# 上传 deploy-script.sh 到服务器后执行
bash deploy-script.sh
```

或者手动执行以下步骤：

### 2.1 安装基础依赖

```bash
sudo apt-get update -y
sudo apt-get install -y curl git nginx certbot python3-certbot-nginx
```

### 2.2 安装 Node.js v22

```bash
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo bash -
sudo apt-get install -y nodejs
node -v  # 确认 v22.x
npm -v
```

### 2.3 安装 PM2

```bash
sudo npm install -g pm2
```

---

## 第三步：上传项目代码

### 方式A：从本地上传（推荐）

```bash
# 在本地终端执行，将项目打包上传
# 先打包项目（排除 node_modules 和 .next）
cd ~/Library/Application\ Support/Claude/hok-world-guide
tar czvf ~/hok-world-guide.tar.gz \
  --exclude='node_modules' \
  --exclude='.next' \
  --exclude='.git' \
  .

# 上传到服务器
scp ~/hok-world-guide.tar.gz ubuntu@你的公网IP:/tmp/

# 在服务器上解压
ssh ubuntu@你的公网IP "sudo mkdir -p /var/www/hok-world-guide && sudo tar xzvf /tmp/hok-world-guide.tar.gz -C /var/www/hok-world-guide/ && sudo chown -R ubuntu:ubuntu /var/www/hok-world-guide"
```

### 方式B：用 Git 拉取（如果代码在 GitHub）

```bash
git clone https://github.com/你的用户名/hok-world-guide.git /var/www/hok-world-guide
```

---

## 第四步：构建项目

```bash
cd /var/www/hok-world-guide
npm install
npm run build
```

> 构建过程约 1-3 分钟，会生成 `.next` 目录。

---

## 第五步：配置 PM2 守护进程

创建 `ecosystem.config.js`：

```javascript
module.exports = {
  apps: [{
    name: 'hok-world-guide',
    cwd: '/var/www/hok-world-guide',
    script: 'node_modules/next/dist/bin/next',
    args: 'start',
    env: {
      NODE_ENV: 'production',
      PORT: 3000
    },
    instances: 1,
    exec_mode: 'fork',
    max_memory_restart: '1G',
    log_date_format: 'YYYY-MM-DD HH:mm:ss Z',
    autorestart: true
  }]
};
```

启动：
```bash
pm2 start ecosystem.config.js
pm2 save
pm2 startup  # 设置开机自启
```

---

## 第六步：配置 Nginx 反向代理

```bash
sudo nano /etc/nginx/sites-available/hok-world-guide
```

写入：

```nginx
server {
    listen 80;
    server_name 你的域名.com www.你的域名.com;

    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_cache_bypass $http_upgrade;
    }

    # 静态资源缓存
    location /_next/static {
        proxy_pass http://localhost:3000;
        proxy_cache_valid 200 365d;
        add_header Cache-Control "public, immutable";
    }

    # 图片缓存
    location /images {
        proxy_pass http://localhost:3000;
        proxy_cache_valid 200 30d;
        add_header Cache-Control "public";
    }
}
```

启用配置：
```bash
sudo ln -s /etc/nginx/sites-available/hok-world-guide /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl restart nginx
```

---

## 第七步：配置 SSL（HTTPS）

```bash
sudo certbot --nginx -d 你的域名.com -d www.你的域名.com
```

按提示操作，Certbot 会自动：
1. 申请免费 SSL 证书（Let's Encrypt）
2. 配置 Nginx 自动跳转 HTTPS
3. 设置证书自动续期

---

## 第八步：验证部署

```bash
# 检查服务状态
curl http://localhost:3000  # 本地应返回HTML
curl -I http://你的域名.com  # 应返回 200 OK

# 检查 PM2
pm2 status
pm2 logs hok-world-guide

# 检查 Nginx
sudo systemctl status nginx
```

---

## 日常维护命令

```bash
# 查看日志
pm2 logs hok-world-guide

# 重启应用
pm2 restart hok-world-guide

# 重新构建后重启
cd /var/www/hok-world-guide && npm run build && pm2 restart hok-world-guide

# 更新代码
cd /var/www/hok-world-guide && git pull && npm install && npm run build && pm2 restart hok-world-guide

# 查看 Nginx 日志
sudo tail -f /var/log/nginx/access.log
sudo tail -f /var/log/nginx/error.log
```

## 备案期间的操作

**备案期间（7-15天），网站不能通过域名访问。**

但你仍然可以：
1. 用服务器IP直接访问测试：`http://服务器IP:3000`
2. 完善网站内容
3. 配置广告位代码（先不上线）

**备案通过后：**
1. 域名解析到服务器IP
2. 申请 SSL 证书
3. 正式上线

5. **世界观** — 游戏背景故事资料整理

## 备案提交步骤

1. 登录 [腾讯云备案系统](https://console.cloud.tencent.com/beian)
2. 选择「新增备案」→「个人备案」
3. 填写主体信息（身份证、手机号、邮箱）
4. 填写网站信息（复制上面的文案）
5. 上传身份证照片（正反面）
6. 用腾讯云APP拍摄免冠照
7. 在线签署承诺书
8. 提交审核

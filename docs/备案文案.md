# 腾讯云 ICP 备案文案指南

> 备案时间：约 7-15 天 | 个人备案只能做非经营性网站

---

## ⚠️ 备案雷区（千万别写）

| ❌ 禁用词 | 原因 |
|----------|------|
| 中国、中华、国家 | 需要国家级资质 |
| 官方、正版、授权 | 涉及官方背书 |
| 王者荣耀、Honor of Kings | 商标侵权 |
| 腾讯、天美 | 商标侵权 |
| 交易、商城、购买、支付 | 经营性网站需企业资质 |
| 广告、推广、营销、代理 | 经营性描述 |
| 博客、论坛、社区 | 个人备案对这类审核更严 |

---

## ✅ 推荐备案方案（3套）

### 方案一：个人笔记/学习记录（通过率最高 ⭐⭐⭐⭐⭐）

```
网站名称：游戏学习笔记
网站域名：hokworld.guide（示例）
网站简介：
  个人游戏学习笔记网站，记录游戏探索过程中的
  心得体会、角色培养经验和地图探索路线，仅供
  个人学习参考使用，不涉及任何商业经营活动。

网站内容说明：
  本网站为个人非经营性网站，主要分享个人在游戏
  过程中整理的角色信息、武器装备数据、地图探索
  路线等学习笔记，内容来源于个人游戏体验整理，
  不涉及任何商业广告或付费服务。
```

**优点：** "学习笔记"是最安全的备案类型，管局最容易通过。

---

### 方案二：游戏经验分享（稍微扩展一点）

```
网站名称：稷下游戏志
网站域名：hokworld.guide（示例）
网站简介：
  个人游戏经验分享网站，记录游戏探索心得、角色
  养成攻略和地图收集路线。所有内容均为个人游戏
  体验整理，供玩家交流参考，不涉及任何商业用途。

网站内容说明：
  本网站为个人非经营性网站，内容主要包括：
  1）游戏角色基础信息整理；
  2）武器装备数据记录；
  3）地图探索路线分享；
  4）个人游戏心得体会。
  所有内容均为原创整理，不涉及官方授权或商业合作。
```

**优点：** "经验分享"也安全，比"笔记"显得内容更丰富。

### 方案三：游戏数据整理（适合内容较多的站）

```
网站名称：王者大陆探索志
网站域名：hokworld.guide（示例）
网站简介：
  个人游戏数据整理网站，收集整理游戏内的角色信息、
  武器装备数据和地图探索内容，方便个人查阅和玩家
  之间的经验交流。本站为纯个人兴趣项目，不从事
  任何经营性活动。

网站内容说明：
  本网站为个人非经营性网站，主要功能：
  - 整理记录游戏内公开的角色基础属性信息
  - 记录武器装备的基础数据
  - 分享地图探索的个人路线
  - 记录游戏剧情和世界观设定
  所有数据均来自个人游戏内体验和公开资料整理，
  不涉及任何官方授权或商业合作行为。
```

**优点：** 提前说明"数据来自公开资料"，避免版权疑虑。

## 🎬 备案拍照/视频注意事项

腾讯云现在大部分情况用**视频核验**（不用幕布拍照了）：

1. **背景**：纯色墙面，不要杂乱
2. **着装**：正常衣服，不要穿制服
3. **动作**：按提示做指定动作（如眨眼、摇头）
4. **念文字**：可能需要念备案号或承诺语
5. **光线**：明亮，不要背光

## ⏱️ 备案时间线

```
Day 1:  提交备案资料
Day 2:  腾讯云初审（1-2个工作日）
        ↓ 初审通过
Day 3:  视频核验（当天完成）
        ↓ 提交管局
Day 4-14: 管局审核（7-10个工作日）
        ↓ 审核通过
Day 15: 收到备案成功短信，获取备案号
```

> 管局审核期间**网站必须关闭**（不能有任何访问），否则会被驳回。
> 建议：备案期间先用服务器IP访问开发和测试，备案通过后再解析域名。

---

## 🔒 备案通过后的操作

1. **网站底部添加备案号**：
   ```html
   <a href="https://beian.miit.gov.cn/" target="_blank">
     粤ICP备XXXXXXXX号
   </a>
   ```

2. **公安备案（可选但建议做）**：
   - 地址：www.beian.gov.cn
   - 时间：ICP备案通过后 30 天内
   - 也需要在底部添加公安备案号

## 💡 提高通过率的技巧

1. **网站名称越朴素越好**："XX笔记"、"XX志" > "XX攻略站"、"XX网"
2. **不要提"服务其他用户"**：个人备案只能服务自己，说"供其他玩家使用"容易被驳回
3. **提前关闭网站**：提交备案前确保域名访问不了（显示403或空白页）
4. **手机号必须本人实名**：管局会核验手机号实名信息是否与身份证一致
5. **地址精确到门牌号**：不要只写省市区，要写完整地址

---

## ❓ 如果被驳回怎么办

常见驳回原因：
- "网站名称不符合要求" → 换成更朴素的名字（如"游戏学习笔记"）
- "网站内容涉及经营" → 删除所有广告、付费、推广相关描述
- "网站可访问且内容不符" → 确保备案期间网站关闭
- "地址不详细" → 补全到门牌号

被驳回后按提示修改，重新提交即可，不收费。

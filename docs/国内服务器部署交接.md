```bash
ssh -i /Users/open/.ssh/id_ed25519_hok ubuntu@2402:4e00:c030:7600:bad6:572c:26a0:0
```

现象：

- TCP 22 端口可连通。
- SSH 连接建立后，服务器没有返回 SSH banner。
- 最终报错：

```text
Connection timed out during banner exchange
Connection to 110.40.181.128 port 22 timed out
```

这说明问题发生在认证之前，不是 key 被拒，也不是账号密码错误。更像是服务器端 `sshd` 卡住、连接数/安全策略限制、或腾讯云侧 SSH 防护异常。

## 建议下一步

1. 登录腾讯云控制台，用 WebShell/VNC/救援终端进入服务器。
2. 检查 SSH 服务：

```bash
sudo systemctl status ssh
sudo journalctl -u ssh -n 100 --no-pager
sudo ss -ltnp | grep ':22'
```

3. 如 SSH 服务异常，重启：

```bash
sudo systemctl restart ssh
```

4. SSH 恢复后，在服务器上直接拉取最新代码并部署：

```bash
cd /var/www/hok-world-guide
git pull origin main
npm install
npm run build
pm2 restart hok-world-guide
pm2 status
```

5. 验证国内站：

```bash
curl -I http://localhost:3000
curl -I https://hokworld.site
```

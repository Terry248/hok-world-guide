## 一、百度联盟重新申请（上次未通过）

### 上次失败原因分析
根据记忆文件，上次申请时填写：
- 网站名称：**王者荣耀世界攻略站**
- 结果：**审核未通过**

**可能原因：**
1. "王者荣耀"是腾讯注册商标，名称涉及商标侵权
2. "攻略站"属于垂直内容站，百度联盟对这类审核较严
3. 网站名称带有商业气息，不够"个人化"

### 重新申请优化方案

#### 方案A：个人游戏笔记（推荐）

| 字段 | 填写内容 |
|------|---------|
| 网站名称 | **游戏学习笔记** |
| 网站描述 | 个人游戏学习笔记，记录游戏探索过程中的角色培养经验、地图探索路线和武器装备数据整理 |
| 网站类别 | 游戏 > 游戏资料 |
| 网站关键词 | 游戏攻略,角色培养,地图探索 |

**优点：** "学习笔记"最符合百度联盟对个人站点的偏好，弱化"攻略站"的商业感。

#### 方案B：游戏数据整理

| 字段 | 填写内容 |
|------|---------|
| 网站名称 | **王者大陆探索志** |
| 网站描述 | 个人游戏数据整理网站，收集整理游戏内的角色信息、武器装备数据和地图探索内容 |
| 网站类别 | 游戏 > 游戏资料 |

**优点：** 不直接使用"王者荣耀"商标，用"王者大陆"替代。

- [ ] 确认网站已备案（✅ 赣ICP备2026010150号）

## 二、腾讯优量汇申请

### 申请条件
- ✅ 网站已备案（满足）
- ✅ 网站内容健康（满足）
- ✅ 无违规内容（满足）

### 申请步骤
1. 访问 https://e.qq.com/ads/ 注册账号
2. 选择「媒体合作」→「网站」
3. 填写网站信息：

| 字段 | 填写内容 |
|------|---------|
| 网站名称 | **游戏学习笔记** 或 **王者大陆探索志** |
| 网站域名 | hokworld.site |
| 网站类型 | 游戏资讯/攻略 |
| 日均PV | 如实填写（新站可先写预估） |
| ICP备案号 | 赣ICP备2026010150号 |

4. 添加广告位代码到网站
5. 等待审核（通常3-7个工作日）

### 腾讯优量汇优势
- 游戏类广告资源丰富（与腾讯游戏生态打通）
- eCPM 较高
- 适合游戏攻略类网站

## 三、Google AdSense 跟进

### 当前状态
- 申请日期：2026-05-09
- 已等待：22天
- 发布商ID：ca-pub-4441684825498793
- 部署平台：Vercel（hok-world-guide.vercel.app）

### 建议操作
1. **登录 AdSense 后台** https://www.google.com/adsense 查看审核状态
2. **检查邮箱**（包括垃圾邮件）是否有 Google 的审核通知
3. **确认 ads.txt 可访问**：
   ```
   https://hok-world-guide.vercel.app/ads.txt
   ```
   应返回：`google.com, pub-4441684825498793, DIRECT, f08c47fec0942fa0`
4. **确认隐私政策页面存在**：`/privacy` 可正常访问
5. **确认网站内容充足**：目前60+页面，应该足够

### 如果审核被拒
常见原因及对策：
- **内容不足** → 继续添加攻略文章
- **流量不足** → 等待自然流量增长或通过社交媒体引流
- **导航不清晰** → 检查网站结构是否合理
- **版权内容** → 确保所有图片/数据有合法来源
